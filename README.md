## Install

First, make sure that install the `stow` command

```
brew install stow
```

Then, clone this repo and run `make stow` to link dotfiles.
```
$ git clone https://github.com/Ackerr/dotfiles.git && cd dotfiles

$ make stow
```

> Remember replace `.gitconfig` email
